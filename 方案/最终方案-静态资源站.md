# Agent时代教程 - 静态资源站方案

## 🎯 产品定位

**一个纯静态的个人知识库 + 资源下载站**
- 用户付费购买卡密
- 输入卡密解锁全站内容
- 随意浏览、下载资料
- 你定期更新，推送到服务器即可

## 🏗️ 技术方案

### 方案A：VitePress + 简单加密（推荐）

**最简单，0成本**

#### 技术栈
- VitePress（静态网站生成器）
- 客户端加密（JavaScript）
- Vercel/Netlify（免费托管）
- 卡密系统（纯前端实现）

#### 工作原理
```
用户访问 → 输入卡密 → 本地验证 → 解锁内容
                ↓
         卡密存储在浏览器localStorage
                ↓
         下次访问自动登录
```

#### 卡密验证方式
```javascript
// 预设的卡密列表（加密存储）
const validKeys = [
  'AGENT2026-A1B2C3',
  'AGENT2026-D4E5F6',
  // ... 你生成的卡密
];

// 用户输入卡密 → 验证 → 解锁
```

#### 优点
- ✅ 完全免费托管
- ✅ 无需后端服务器
- ✅ 部署超简单（git push即可）
- ✅ 速度极快
- ✅ 你随时更新内容

#### 缺点
- ⚠️ 卡密可能被分享（但可以限制使用次数）
- ⚠️ 无法统计用户行为

---

### 方案B：VitePress + Cloudflare Workers（进阶）

**稍微复杂，但更安全**

#### 技术栈
- VitePress（前端）
- Cloudflare Workers（卡密验证API）
- Cloudflare KV（卡密数据库）
- 免费额度：每天10万次请求

#### 工作原理
```
用户输入卡密 → 请求API验证 → 返回Token → 解锁内容
                    ↓
            记录使用次数/设备数
```

#### 优点
- ✅ 卡密更安全（服务端验证）
- ✅ 可以限制设备数/使用次数
- ✅ 可以统计数据
- ✅ 仍然免费（Cloudflare免费额度够用）

#### 缺点
- ⚠️ 需要写一点后端代码（很简单）

## 🔐 付费墙实现

### 内容分级

#### 公开内容（无需登录）
- 首页介绍
- 定价说明
- 部分免费教程（引流）

#### 付费内容（需要卡密）
- 完整教程文档
- 项目源码下载
- 技能包下载
- 真实案例详解

### 实现方式

```vue
<!-- PaywallGuard.vue -->
<template>
  <div v-if="!isUnlocked">
    <div class="paywall">
      <h2>🔒 此内容需要解锁</h2>
      <p>购买后获取卡密，即可访问全部内容</p>
      <button @click="showLogin">输入卡密</button>
      <a href="/pricing">查看定价</a>
    </div>
  </div>
  <div v-else>
    <slot /> <!-- 显示实际内容 -->
  </div>
</template>
```
